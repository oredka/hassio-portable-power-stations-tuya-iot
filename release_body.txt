## Release 2.4.1

This release fixes critical UI issues and rebrands the integration for better clarity.

### Fixed
- **500 Internal Server Error**: Fixed a crash when opening the integration's configuration (Options Flow).
- **Integration Name**: Renamed to "Tuya IoT Smart Portable Power Stations for Home Assistant" for better identification.

### Installation
Update via HACS or by copying the `custom_components/tuya_iot_power_stations` folder to your Home Assistant installation.
