# Troubleshooting

## Іконка інтеграції не відображається

Якщо іконка інтеграції не з'являється в Home Assistant, спробуйте наступне:

### Метод 1: Очистити кеш браузера
1. Натисніть `Ctrl+Shift+Delete` (або `Cmd+Shift+Delete` на Mac)
2. Виберіть "Зображення та файли в кеші"
3. Очистіть кеш
4. Перезавантажте сторінку Home Assistant (`Ctrl+F5` або `Cmd+Shift+R`)

### Метод 2: Використати режим інкогніто
Відкрийте Home Assistant у режимі інкогніто/приватного перегляду - якщо іконка з'являється там, проблема точно в кеші.

### Метод 3: Перезапустити Home Assistant
1. Налаштування → Система → Перезапустити
2. Після перезапуску очистіть кеш браузера

### Метод 4: Перевстановити інтеграцію
1. Видаліть інтеграцію (Налаштування → Пристрої та служби → 2E Power Stations → Видалити)
2. Очистіть кеш браузера
3. Перезапустіть Home Assistant
4. Додайте інтеграцію знову

## Примітки

- Іконка може з'явитися не одразу після встановлення
- Home Assistant кешує іконки інтеграцій
- Після оновлення інтеграції завжди очищайте кеш браузера
- Іконка показується в списку інтеграцій, а не обов'язково на картках пристроїв

## LED Mode

### Підтримувані режими
2E SYAYVO-BP2400_D підтримує наступні режими LED підсвічування:
- **Off** - вимкнено
- **10%** - 10% яскравості
- **30%** - 30% яскравості
- **50%** - 50% яскравості
- **100%** - 100% яскравості
- **SOS** - режим SOS

**Примітка:** Режим "Breathing" (дихання) не підтримується цією моделлю та був видалений з опцій у версії 1.3.0.

### Якщо LED Mode не працює

1. Перевірте що ви використовуєте підтримуваний режим (див. список вище)
2. Перевірте логи Home Assistant (Налаштування → Система → Логи)
3. Знайдіть повідомлення `Failed to set LED Mode`
4. Створіть issue на GitHub з логами: https://github.com/oredka/2e-power-stations/issues

## Проблеми з підключенням до Tuya Cloud

Якщо інтеграція не може підключитися до Tuya Cloud:

1. Перевірте що Access ID, Access Secret та Device ID правильні
2. Переконайтеся що пристрій підключений до Tuya IoT Platform:
   - Увійдіть на https://iot.tuya.com
   - Cloud → Development → ваш проєкт
   - Devices - пристрій має бути в списку
3. Перевірте що пристрій авторизовано через App Account:
   - Cloud → Development → Link Tuya App Account
   - Увійдіть через Smart Life або Tuya Smart app
4. Переконайтеся що API активовано:
   - Service API → IoT Core → Subscribe (безкоштовно)
